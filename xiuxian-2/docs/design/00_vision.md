# 00 愿景

## 产品定位
- 类型：桌面底部驻留的修仙桌宠 + 轻量养成。
- 核心驱动：读取用户真实键盘/鼠标操作次数作为修炼进度。
- 引擎与语言：Godot 4 + C#。
- 运行形态：常驻后台，用户工作学习时自然积累进度。

## 核心体验
1. 做事即修仙：用户在日常打字、点击中持续成长。
2. 低打扰陪伴：桌宠在桌面底部活动，不遮挡主工作区。
3. 明确可控：进度规则透明，玩家能通过策略分配资源。

## V1 范围
- 包含：
- 全局键鼠计数与活动点转换。
- 境界成长与突破。
- 桌面底栏展示 + 书卷式系统面板。
- 灵宠心情与互动加成。
- 本地存档、离线结算、基础防刷。
- 不包含：
- 联网排行/社交。
- 复杂战斗和大地图探索。
- 账号云同步。

## 隐私与合规原则
- 只统计事件数量与频率，不记录按键内容。
- 不上传原始输入数据，默认仅本地处理。
- 提供一键暂停采集和状态可视化提示。

## V1 目标指标
- D1 留存 >= 40%。
- 首日 60-90 分钟累计使用可达第 3 境。
- 70% 用户能理解“输入行为 -> 修炼收益”的映射关系。
